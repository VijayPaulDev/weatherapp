# Weather App 

A simple **React + TypeScript + Tailwindcss + Axios + Vite** weather app that fetches weather information from the OpenWeatherMap API.  
Displays City and the Country, temperature, description, humidity, wind speed, pressure, and 'feelslike' temp for any city.

## Features

- Search weather by city
- Displays:
  - Temperature (Â°C)
  - Weather description
  - Humidity, Pressure, Wind speed and 'Feels like' temperature
- Weather icons
- Uses TypeScript for type safety
- Uses TailwindCSS styling
- Uses axios for http requests
- Built with Vite development server

## Installation

1. Clone the repo:

-   git clone https://github.com/VijayPaulDev/weatherapp.git 

2. Install dependencies:
- npm 
- react
- typescript
- tailwindcss
- axios

3. Create a .env file in the root and add your OpenWeatherMap API key:
  VITE_WEATHER_API_KEY= **your api key here**
